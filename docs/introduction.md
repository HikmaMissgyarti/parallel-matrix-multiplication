# Introduction

Parallel computing is a method of solving computational problems by dividing a task into smaller parts that can run simultaneously.

Many modern computers have multiple CPU cores, allowing several processes to execute at the same time. By taking advantage of these resources, computational tasks can be completed faster compared to traditional sequential execution.

Matrix multiplication is a common operation in computer science, data processing, artificial intelligence, and scientific computing. When the matrix size becomes large, the computation requires a significant amount of time and processing power.

In this project, matrix multiplication is used as a case study to compare sequential execution and parallel execution using Python's multiprocessing library. The performance of both approaches is measured and analyzed to observe the benefits of parallel computing.
