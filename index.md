# Parallel Matrix Multiplication

## Overview

This project compares sequential and parallel matrix multiplication using Python multiprocessing.

## Features

* Sequential Matrix Multiplication
* Parallel Matrix Multiplication
* Performance Comparison
* Speedup Analysis

## Result

| Method     | Time    |
| ---------- | ------- |
| Sequential | 24.48 s |
| Parallel   | 9.60 s  |

**Speedup: 2.55x**
